# LMS Frontend

Frontend application for the **Learning Management System (LMS)** built with **React**, **Material UI**, and **Vite**.  
This application provides the user interface for administrators to manage students, notices, and authentication.

---

## 🚀 Tech Stack

- React (Vite)
- Material UI (MUI)
- Redux Toolkit
- Axios
- React Router
- Prettier
- Husky (Git Hooks)

---

## 📦 Prerequisites

Make sure the following are installed:

- Node.js (v16 or higher)
- npm or yarn
- Backend server running

---
