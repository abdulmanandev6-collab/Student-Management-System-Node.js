This is a School leabribg Management system created with React, Material Ui, Express js.
The system provides tools for managing students, notices, authentication, and generating student reports.

The LMS provides administrators with a centralized platform to:

Manage students

Publish notices

Handle authentication and authorization

Generate student PDF reports

Maintain structured academic data



Frontend

React

Vite

Material UI (MUI)

Redux Toolkit

Axios

React Router

Backend

Node.js

Express.js

SQlite

JWT Authentication


Prerequisites

Before running the project ensure you have:

Node.js (v16 or higher)

PostgreSQL (v12 or higher)

npm or yarn

Git


Installation & Setup
git clone <repository-url>
cd project-root

Backend
cd backend
npm install
npm start


Frontend
cd frontend
npm install
npm run dev
